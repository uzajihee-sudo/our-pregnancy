# Vercel Deployment

이 앱은 빌드 과정이 없는 정적 사이트라서 `Vercel`에 바로 올리면 됩니다.

## 가장 단순한 방법

1. GitHub에 새 저장소를 하나 만듭니다.
2. 이 폴더 파일 전체를 그 저장소에 올립니다.
3. `https://vercel.com/new` 에서 GitHub 저장소를 연결합니다.
4. Framework Preset은 `Other` 그대로 둡니다.
5. Build Command와 Output Directory는 비워둔 채 배포합니다.

## 배포 후 동작

- 주소를 남편과 공유하면 같은 화면을 같이 볼 수 있습니다.
- 체크 완료 상태와 마지막으로 본 업데이트 상태는 각자 브라우저에 따로 저장됩니다.
- 새 혜택을 반영해 다시 배포하면 앱 안의 `업데이트 확인` 버튼으로 최신 배포본을 다시 읽을 수 있습니다.
